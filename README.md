# SOC Home Lab - Wazuh + Sysmon

## Overview
This project demonstrates a Security Operations Center (SOC) Home Lab using Wazuh SIEM, Sysmon, Windows 10, and Ubuntu Server.

## Lab Setup
- Ubuntu Server (Wazuh Manager)
- Windows 10 (Wazuh Agent)
- Sysmon
- VirtualBox

## Tools
- Wazuh 4.12
- Sysmon
- Windows 10
- Ubuntu Server
- VirtualBox

## Skills
- SIEM
- Threat Hunting
- Log Analysis
- Windows Event Monitoring
- Sysmon Monitoring
